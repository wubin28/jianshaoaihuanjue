● SEO (Search Engine Optimization)

  定义

  搜索引擎优化 - 通过优化网站内容和结构，提高在搜索引擎（如Google、百度）中的排名，让用户更容易找到你的网站。

  特点

  - 依赖搜索引擎爬虫: 需要搜索引擎能够读取和理解你的网页内容
  - 需要完整的HTML: 爬虫主要读取HTML，不执行JavaScript（或执行有限）
  - 长期投资: 效果不是立竿见影，需要持续优化

  优势

  1. 免费流量: 自然搜索带来的访问不需要付费广告
  2. 精准用户: 主动搜索的用户通常有明确需求
  3. 长期收益: 排名稳定后可持续带来流量
  4. 品牌信任: 高排名提升品牌可信度

  劣势

  1. 见效慢: 通常需要数月才能看到效果
  2. 竞争激烈: 热门关键词排名困难
  3. 算法变化: 搜索引擎算法更新可能影响排名
  4. 需要持续维护: 不能一劳永逸

  适用场景

  强烈需要SEO

  - 📰 内容型网站: 博客、新闻、媒体
  - 🛒 电商网站: 产品页面、分类页
  - 🏢 企业官网: 公司介绍、服务展示
  - 📚 文档/教程: 技术文档、学习资源

  不需要SEO

  - 🔒 后台管理系统: 需要登录才能访问
  - 💬 社交应用: 内容由用户生成，无需搜索引擎导流
  - 🎮 在线工具: 用户直接访问，不通过搜索
  - 📱 原生App: 不在Web上运行

  为什么Next.js对SEO友好？

  // 服务端渲染 - SEO友好 ✅
  // 搜索引擎爬虫收到的HTML:
  <div>
    <h1>你好，世界</h1>
    <p>这是文章内容...</p>
  </div>

  // 纯客户端渲染 - SEO不友好 ❌
  // 搜索引擎爬虫收到的HTML:
  <div id="root"></div>
  <script src="app.js"></script>
  // 内容需要等JavaScript执行才能生成，爬虫可能看不到

  简单记忆:
  - SEO = 让Google能"看懂"你的网站内容
  - 服务端渲染 = 给Google一份完整的"菜单"
  - 客户端渲染 = 给Google一张"白纸"，让它自己画
